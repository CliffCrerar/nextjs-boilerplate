## About

## Boiler ingredients

1. Next.js
2. React
3. Materialize CSS
4. Bootstrap4 Utility Classes

## Theming

The project comes with 3 standard themes:
1. default ( Material UI theme )
1. blue-deeporange
2. indigo-blue
3. pink-indigo

Create new themes by placing `.css` files create with [material design pallete](https://www.materialpalette.com/)
